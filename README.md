## Running the project

### Run

**Install dependencies**

```
yarn
```

**Install IOS Pods**

```
cd ios && pod install
```

**Run IOS**

```
yarn start | yarn ios
```

**Run Android**

```
yarn start | yarn android
```

## Running the tests

Use **yarn test** to execute the jest tests.

```cl
yarn test
```
